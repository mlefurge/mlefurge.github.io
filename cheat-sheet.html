SQL

Select an entire table

    SELECT * FROM * table_name;

Select column from table

    SELECT column_name1,column_name2 FROM table_name;

Select columns and sort by ascending or descending

    SELECT column_name1,column_name2 FROM table_name ORDER BY column_name2 DESC;
    Note: ORDER BY defaults to ascending order

Select items in a column based on criteria

    SELECT column_name1 from table_name WHERE column_name2 operator value;
    Note: Operate value can equal: =, <> (not equal), <, >, <=, >=, BETWEEN, LIKE, IN.

Select items in a column that contain certain characters

    SELECT column_name1 from table_name WHERE column_name1 LIKE 'desired_text'
    Note: Wildcards: % = substitute for zero or more characters, _ = substitute for single character
